# SeSAC Python Full Stack Practice

This repository contains **practice materials and example code** created while studying in the  
**SeSAC (Seoul Software Academy) Bootcamp**,  
as part of the **Python Flask Full Stack** course.

The project is a **personal learning archive**, covering a wide range of topics from basic web technologies to backend development with Python and Flask.

---

## Overview

This repository was created to:

- Practice core web development concepts
- Understand frontend and backend integration
- Learn Python-based backend development using Flask
- Review and organize bootcamp lecture materials

All code and files were written as part of hands-on exercises during the SeSAC bootcamp.

---

## Topics Covered

### Frontend
- HTML basics
  - Tags, forms, tables, semantic elements
- CSS basics
  - Layout and styling
- JavaScript fundamentals
  - DOM manipulation
  - Event handling

### Backend
- Python fundamentals
- Flask framework
  - Routing
  - Templates
  - Request / Response handling
- Basic server-side logic

### Full Stack Concepts
- Client–server communication
- Form handling
- Data flow between frontend and backend
- Project structure and organization
